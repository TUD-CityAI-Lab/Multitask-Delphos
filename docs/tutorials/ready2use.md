# Ready to Use: Apollo Integration

One of the key features of Delphos is its ability to directly output model specifications that can be run in the popular R package **Apollo**. 

This tutorial shows how to take a specification generated by Delphos and estimate it using Apollo.

## Generating the Apollo Script

Once Delphos has completed its exploration phase, you can select any model from the Pareto front and export it as an Apollo R script.

```python
# Assuming 'agent' has already explored the dataset
pareto_front = agent.get_pareto_front()

# Get the model with the best BIC score
best_model = pareto_front.get_best_by_metric("BIC")

# Export to Apollo
best_model.export_to_apollo(
    filepath="apollo_model.R",
    dataset_path="data/my_dataset.csv",
    model_name="Delphos_Best_BIC"
)
```

## Running the Script in R

The generated `apollo_model.R` script is fully self-contained. It includes:
- Data loading.
- Parameter initialization (starting values).
- The utility functions defined by Delphos.
- The Apollo probabilities and log-likelihood functions.
- The command to run the estimation and output the results.

You can run this script directly in R or via the command line:

```bash
Rscript apollo_model.R
```

## Example Generated Script

Here is an example of what the generated R script looks like:

```R
# --- Generated by Delphos ---
library(apollo)

# Initialize Apollo
apollo_initialise()

# Set core controls
apollo_control = list(
  modelName  = "Delphos_Best_BIC",
  modelDescr = "Auto-generated model by Delphos",
  indivID    = "ID"
)

# Load data
database = read.csv("data/my_dataset.csv")

# Define parameters
apollo_beta = c(
  asc_car   = 0,
  asc_train = 0,
  b_time    = 0,
  b_cost    = 0
)

# Define fixed parameters
apollo_fixed = c("asc_car")

# Define probabilities function
apollo_probabilities = function(apollo_beta, apollo_inputs, functionality="estimate"){
  apollo_attach(apollo_beta, apollo_inputs)
  on.exit(apollo_detach(apollo_beta, apollo_inputs))
  
  P = list()
  
  V = list()
  V[["car"]]   = asc_car + b_time * time_car + b_cost * cost_car
  V[["train"]] = asc_train + b_time * time_train + b_cost * cost_train
  
  mnl_settings = list(
    alternatives = c(car=1, train=2),
    avail        = list(car=avail_car, train=avail_train),
    choiceVar    = choice,
    V            = V
  )
  
  P[["model"]] = apollo_mnl(mnl_settings, functionality)
  P = apollo_prepareProb(P, apollo_inputs, functionality)
  return(P)
}

# Estimate model
model = apollo_estimate(apollo_beta, apollo_fixed, apollo_probabilities, apollo_inputs)

# Output results
apollo_modelOutput(model)
```
