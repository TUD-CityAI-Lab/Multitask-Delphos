# Datasets Overview

The Delphos ecosystem includes a dedicated repository for datasets: **`delphos-datasets`** (hosted at `transport-choice-datasets`).

This repository contains a standardized collection of transportation choice datasets used for training, inference, benchmarking, and reproducibility of the Delphos framework.

By standardizing these datasets, we ensure that the multi-task reinforcement learning agent can learn robustly across different choice contexts.

## Contents

- Raw and processed datasets (CSV format)
- Dataset Schemas (JSON format)
- Variable dictionaries
- Preprocessing pipelines
